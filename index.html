Guessing Game - JavaScript

What the JavaScript on this page is doing.

Below are the steps and their explanations.

  1. The rightAnsCounter variable stores how many time the user answered correctly, it starts out at 0.

  2. The prompt("what is your name") method prompts the user for his/her name and stores it in the userName variable.

  3. The while loop is used to require the user to input a name.

  4. The alert("Hi " + userName + "! Nice to ...) method takes the user's name from the userName variable and concatenate the user's name with a greeting string, then displays the message to the user in a an alert box.

  5. The First guessing game question is asked with the prompt() method, then stores the user's answer to answer1 variable.

  6. The while loop is then used to ensure the user only inputs yes or no. If other than yes or no is inputed, the user is alerted with "Invalid answer!..." and then prompts the user to answer the question again.

  7. The toLowerCase() method is used to convert the user's answer to all lowercase for proper comparison.

  8. The if else statement then evaluate the user's answer to the question. If user's answer is correct, 1 is added to the rightAnsCounter. This process repeats for question 2 and question 3.


  10. The last alert() method tells the user how many questions he/she got right by concatenating the rightAnsCounter to "to correct answers out of 3 questions".
